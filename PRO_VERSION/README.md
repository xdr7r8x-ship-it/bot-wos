# Dream Memory PRO

Professional overlay for Dream Memory game in Whiteout Survival.

## Features

- **Multi-threaded**: Capture and detection run in background threads
- **High Performance**: Optimized for 60fps capture
- **Color Detection**: No AI needed
- **Lightweight**: Minimal dependencies

## Requirements

- Windows 10/11
- Python 3.8+
- BlueStacks (or other Android emulator)

## Installation

1. Run `START.bat`
2. Wait for installation to complete
3. Application will auto-start

## Controls

| Key | Action |
|-----|--------|
| F8 | Toggle overlay visibility |
| F9 | Toggle monitoring on/off |
| ESC | Exit |

## Manual Start

```powershell
pip install opencv-python pillow PyQt6 pywin32 numpy
python main.py
```
